Snippet Cheatsheet

this is a cheatsheet for different syntax for CSS code. Select any snippet below and it'll automatically select all of the code for you to copy

Flexbox Row

Use these three properties to creat a Flexbox row layout.

.row { display: flex; flex-direction: row; flex-wrap: wrap; }

Flexbox Column

Use this to create a Flexbox column layout.

.column { display: flex; flex-direction: column }

CSS Grid layout

Build a 12-column layhout using CSS Grid

.grid { display" grid; width: 100%; grid-template-columns: repeat(12, 1fr); }
